## Installation

You will need to create an .env file and populate..

`DB_USER= ''`

`DB_PASSWORD= ''`

`DB_NAME='employee_tracker_db'`

Create the Database using schema.sql

`CREATE DATABASE employee_tracker_db;`

Install packages

`NPM install`

Run the application

`Node index.js`
